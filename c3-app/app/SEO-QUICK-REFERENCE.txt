================================================================================
                            INDEXNOW SETUP
================================================================================

1. Get Key
   └─ Visit: https://www.indexnow.org
   └─ Click: Get your key
   └─ Enter: Domain & email
   └─ Copy: 40-character key

2. Store Key
   └─ .env.local: VITE_INDEXNOW_KEY=key
   └─ File: public/.well-known/indexnow
   └─ Verify: https://yourdomain.com/.well-known/indexnow

3. Auto-Submit
   └─ Add component: <IndexNowTracker />
   └─ Automatically submits on page load
   └─ Monitor console for logs

4. Manual Submit
   └─ Use: const { submitUrl } = useIndexNow()
   └─ Call: submitUrl('https://...')
   └─ For: New/updated content

5. Batch Submit
   └─ Use: submitBatchToIndexNow(urls)
   └─ Max: 10,000 URLs per submission
   └─ For: Publishing multiple pages

================================================================================
                       TROUBLESHOOTING QUICK FIX
================================================================================

Issue: Meta tag not detected
  → Hard refresh (Ctrl+Shift+R)
  → Check .env.local has correct code
  → Rebuild: npm run build
  → Redeploy and wait 24 hours

Issue: Pages not indexed
  → Verify sitemap.xml accessible
  → Check robots.txt doesn't block paths
  → Submit sitemap manually in console
  → Wait 48-72 hours

Issue: IndexNow not working
  → Verify .well-known/indexnow exists
  → Check HTTPS enabled
  → Verify key is 40 characters
  → Test with curl command

Issue: TypeScript errors
  → Verify src/vite-env.d.ts exists
  → Restart VS Code
  → Delete node_modules, npm install
  → Run: npm run lint

Issue: Build fails
  → Check all components imported correctly
  → Verify no syntax errors
  → Check .env.local variables exist
  → Run: npm run lint --fix
